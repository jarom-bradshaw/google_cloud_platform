You need a separate account from ones using AWS or Azure to access the workspace via https://accounts.gcp.databricks.com/

1. [Subscribe Databricks](https://console.cloud.google.com/marketplace/product/databricks-prod/databricks?utm_source=databricks&utm_medium=website&utm_content=google-cloud-free-trial&utm_campaign=db-gcptrialpage) on Marketplace on a right account connected to a right project/billing account with credits.
2. Create a workspace (https://accounts.gcp.databricks.com).
3. Do your project.
4. Delete the workspace on Databricks and the project on GCP.
